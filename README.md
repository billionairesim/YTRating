# YT Rater

A simple, minimalistic app to rate YouTube videos on custom parameters.

## Features

- Add videos with a title and custom thumbnail image
- Create your own rating parameters (e.g. Content Quality, Audio, Editing)
- Rate each video on a 5-star scale with half-star precision
- Auto-calculated average score per video
- Built-in analysis showing verdict, best and weakest areas
- All data saved locally in your browser
- Deploy anywhere as a static site

## Getting Started

### Install

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

## Tech Stack

- React
- Vite
- Pure CSS
- localStorage for persistence
